## ⏳ ОСТАЛОСЬ СДЕЛАТЬ:

### **ТОЛЬКО 1 ДЕЙСТВИЕ: Создать лист "Purchases" в Google Sheets** (1 минута)

#### **Шаг 1: Откройте таблицу**
```
https://docs.google.com/spreadsheets/d/1VLb3Z59dG7oAT4PV-sRXA2smHyeWzljiO8jl4ynK0M4/edit
```

#### **Шаг 2: Создайте лист**
1. Нажмите **"+"** (плюс) внизу экрана рядом с вкладками листов
2. Переименуйте новый лист в **`Purchases`** (точное название!)
   - ПКМ по вкладке → "Переименовать"
   - Введите: `Purchases`

#### **Шаг 3: Добавьте заголовки**
В ячейку **A1** вставьте (это 8 колонок через табуляцию):
```
ID	CategoryID	Name	Amount	Date	Description	Supplier	CreatedAt
```

**Как правильно вставить:**
- Скопируйте строку выше (включая невидимые символы табуляции)
- Кликните на ячейку **A1**
- Нажмите **Ctrl+V** (или Cmd+V на Mac)
- Заголовки должны разойтись по колонкам A, B, C, D, E, F, G, H

#### **Шаг 4: Проверьте**
После вставки должно получиться:
```
| A: ID | B: CategoryID | C: Name | D: Amount | E: Date | F: Description | G: Supplier | H: CreatedAt |
```

## 🎉 ВСЁ! Система готова!

После создания листа "Purchases":

1. Перейдите `/admin/settings` → 🏷️ **Категории закупок**
   - Там уже видны 5 категорий
   - Можете добавлять свои

2. Перейдите `/admin/finances` → 📦 **Закупки**
   - Нажмите "+ Добавить закупку"
   - Выберите категорию из списка
   - Заполните форму
   - Закупка сохранится в Google Sheets

3. Вкладка 📊 **Обзор** покажет:
   - 💰 Доход
   - 💳 Карта
   - 🧾 Налог
   - 📦 **Расходы** ← новая карточка!
   - 💚 Чистыми (доход - налог - расходы)
